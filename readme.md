# HowdyHack

App I build for HowdyHack 24 (:

## Config

Add a `./src/config.json` file with the following content. Obviously use your own api keys

```json
{
    "openAIConfig": {
        "apiKey": "sk-proj-<...>"
    },
    "firebaseConfig": {
        "apiKey": "<...>",
        "authDomain": "<...>.firebaseapp.com",
        "projectId": "<...>",
        "storageBucket": "<...>.appspot.com",
        "messagingSenderId": "<....>",
        "appId": "<...>"
    }
}
```
